Theme Cards Tryout

Description: Huge Theme - based on a modular framework.
Demo Site: https://about-hugo.github.io/theme-cards-tryout
Source Code: https://github.com/about-hugo/theme-cards-tryout
GH Actions: actions
Theme Home Page: https://github.com/hbstack/theme-cards

This is a theme based on HBStack. HBStack is is a modular framework for building Hugo pages, see https://hbstack.dev/ Both the framework and theme are massive and seem well-designed and powerful; before starting a project with it some time would be necessary to explore and evaluate this framework.

Theme home page: https://theme-cards.hbstack.dev/
