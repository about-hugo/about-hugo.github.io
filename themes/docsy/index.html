Docsy Theme Tryout

Demo Site: https://about-hugo.github.io/theme-tryout-docsy/
Source Code: https://github.com/about-hugo/theme-tryout-docsy
GH Actions: actions
Theme Home Page: https://github.com/google/docsy

Docsy is a comprehensive documentation theme built by Google. It includes a blog, makes use of hugos tagging system and provides github links to edit the page.

It can be installed as a module and comes with a comprehensive documentation.
