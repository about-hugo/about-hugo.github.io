Showtime Website

Description: IMI Showtime Web Site
Live Site: https://showtime.f4.htw-berlin.de
Source Code: https://github.com/htw-imi-showtime/showtime-website
GH Actions: actions
Theme Home Page: https://github.com/htw-imi-showtime/showtime-hugo-theme

The IMI Showtime Website was created during the pandemic to replace the Showtime Fair.

Now, as the showtime is back to it’s original format, it will will be continued to supplement the showtime as event home page and project archive.

The Showtime Website was initiated and supervised by B. Kleinen and created by Tobi Hofmann (Design) and Luis Hankel (Development), see https://showtime.f4.htw-berlin.de/imprint/
Deployment: The Showtime Website is deployed by github actions. It is automatically deployed to various Preview- and Staging Sites
Overview Sites: https://htw-imi-showtime.github.io/
Production Deployments: are triggered by github releases (on the)
